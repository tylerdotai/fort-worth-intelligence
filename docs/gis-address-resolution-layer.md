# GIS / Address Resolution Layer

## Why this layer matters

The best wedge for Fort Worth intelligence is probably address-first.

Most people do not want an ontology.
They want an answer for a place.

## Core sources

### Fort Worth MapIt
- Canonical source: `https://mapit.fortworthtexas.gov`
- Type: GIS / map portal
- Role: primary city geospatial discovery surface

### TxDOT Open Data
- Canonical source: `https://gis-txdot.opendata.arcgis.com/`
- Type: GIS / API
- Role: roads, bridges, transportation context

### HIFLD / federal infrastructure overlays
- Type: GIS / bulk
- Role: critical infrastructure context

### Census boundaries and ACS context
- Type: API / boundary data
- Role: demographic and district context

## Desired output for one address
- parcel context
- city district
- county context
- ISD
- transit authority / access context
- tax / appraisal bodies
- utility / special district overlays
- legislative / agenda relevance by district

## Strategic value

Address resolution is the point where multiple public systems stop being a pile of sites and start becoming intelligence.
